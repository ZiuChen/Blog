        剪贴板

        ✨ 记录剪贴板历史并保存到本地，支持文字/图片/文件。

        ☁️ 数据迁移与同步指南 🚚 更新日志

        # 🔰 开始使用

        首次安装需要设置“跟随主程序同时启动”

        • ✅ 监听剪贴板并持续将新内容更新到本地磁盘
        • ✅ 鼠标左键 复制剪贴板历史并粘贴
        • ✅ 鼠标右键 将剪贴板历史复制到剪贴板
        • ✅ 使用 Tab 键切换分类，使用 / Ctrl+F Ctrl+L 聚焦搜索框
        • ✅ 跟随系统设置的深色模式
        • 🎁 键盘 ↑ ↓ 可以选中历史记录，按下回车直接粘贴选中内容
        • 🎁 支持多选历史记录并粘贴
        • 🎁 自定义响应按键行为、自搭建多端同步服务 ···

        # 📚 安装方式

        • 官方插件市场安装
        • 离线插件安装：百度网盘 (opens new window)
