## 🔰 开始使用

**服务器不会保留处理的数据, 但请避免使用此功能处理敏感数据**

- ✅ 支持选中文本后通过超级面板直接进入分词
- ✅ 支持直接读取剪贴板文本分词
- ✅ 支持选词后一键合并复制/跳转翻译
- ✅ 插件功能依赖服务器资源 普通用户每日有免费额度
- 🎁 可以通过插件内入口获取更多额度
- 🎁 与`超级剪贴板`插件集成，一次购买 多处使用
