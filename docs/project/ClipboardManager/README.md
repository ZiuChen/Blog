## 🔰 开始使用

**首次安装需要设置“跟随主程序同时启动”**

- ✅ 监听剪贴板并持续将新内容更新到本地磁盘
- ✅ `鼠标左键` 复制剪贴板历史并粘贴
- ✅ `鼠标右键` 将剪贴板历史复制到剪贴板
- ✅ 键盘 `↑` `↓` 选中历史记录，按下回车直接粘贴选中内容
- ✅ 使用 `Tab` 键切换分类
- ✅ 插件内按下任意键聚焦搜索框（或使用 `Ctrl+F` `Ctrl+L`）
- ✅ 跟随系统设置的深色模式
- 🎁 插件功能自定义、自搭建多端同步服务 ···
