# 快捷键一览

粗体 `Cmd-B`

斜体 `Cmd-I`

链接 `Cmd-K`

图片 `Shift-Cmd-I`

代码 `Shift-Cmd-K`

代码块 `Shift-Cmd-C`

无序列表 `Shift-Cmd-U`

有序列表 `Shift-Cmd-O`

## Markdown 语法

一级标题 `# 标题`

二级标题 `## 标题`

三级标题 `### 标题`

粗体 `**粗体文本**`

斜体 `*斜体文本*`

引用 `> 引用文本`

链接 `[链接描述](url)`

图片 `![alt](url "图片描述")`

代码 ``代码``

代码块 ` ```编程语言↵`

无序列表 `- 项目`

有序列表 `1. 项目`

分割线 `---`

删除线 `~~文本~~`

任务列表 `- [ ] 待办事项`

行内公式 `$公式$`

块级公式 `$$↵公式↵$$`

Mermaid图表 ` ```mermaid`
